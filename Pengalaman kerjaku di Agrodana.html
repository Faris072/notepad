Pengalaman Kerjaku di
PT. Agrodana Futures Surabaya

Oke jadi saya disini akan berbagi pengalaman kerjaku di PT. Agrodana Surabaya. Disini saya akan menjelaskan detail tentang dari awal perekrutan sampai saya lepas training di PT. Agrodana Surabaya dan sistem kerjanya seperti apa. Saya membagi pengalaman kerja saya jujur dari pengalaman saya sendiri sesuai yang saya alami, Karena hal ini sangat penting untuk dipublikasikan sistem kerjanya seperti apa supaya kalian tau dan siap saat bekerja disana. Baiklah mari kita mulai pembahasannya.


1.Perekrutan

Kita mulai dari proses perekrutan. Ketika saya berangkat dari kota saya ke surabaya, saya mendapatkan lowongan tersebut dari media sosial facebook yang membagikan lowongannya di kolom komentar seseorang. Di poster lowongan tersebut sangat bermacam-macam posisi yang di butuhkan mulai dari staff admin, HR, OB, dan lain sebagainya. Namun, sayangnya disitu tidak ada nama perusahaan yang tertera di poster tersebut. Pada saat itu saya juga mendapat panggilan dari perusahaan lain tapi setelah saya pikir-pikir perusahaan yang lain tersebut bergerak di bidang keuangan jadi saya kurang berminat bekerja disitu. Setelah saya tanya-tanya mengenai pekerjaan yang saya temukan tersebut, mereka menamai perusahaan mereka dengan nama PT. Agrogroub dan katanya mereka bergerak di bidang perdagangan hasil bumi. Oke lah dari sini saya mulai tertarik dengan pekerjaan ini karena saya juga mencari alamat kantor melalui Google Maps dan kantor tersebut terlihat menjanjikan dan tidak mencurigakan.

Ketika saya sudah sampai di rumah dan saya membuat lamaran, saya bertanya lagi mengenai kerja di sana untuk memastikan. saya waktu itu bertanya mengenai sistem kerja di sana. karena sebelumnya saya mencari artikel di Google dan katanya kalau perusahaan Futures itu bergerak di bidang jasa keuangan, karena waktu itu saya hanya di beri tau kalau perusahaan ini Agrogroub maka saya merasa bahwa peruahaan ini bukan bergerak di bidang keuangan. Tapi, setelah saya tanya apakah perusahaan ini bergerak di bidang jasa keuangan, mereka menjawab kalau perusahaan ini bergerak di bidang jasa keuangan. Di pikiran saya merasa kalau ini adalah kantor yang menangani hasil bumi. Ya lumayan lah bisa bekerja di kantor. Lalu saya bertanya lagi apakah nanti saya harus mencari nasabah?. Jawaban mereka tergantung posisi yang saya lamar. Lalu saya bertanya lagi kalau saya belum punya pengalaman kerja apakah bisa melamar disini? Jawaban mereka iya boleh nanti di ajari dari awal. Disini saya mulai senang karena jarang sekali perusahaan besar menawarkan pekerjaan bagi orang yang belum berpengalaman. Tapi saya juga ada sedikit keraguan karena proses masuk saya begitu mudahnya tapi berniat untuk mencobanya dulu karena kalau saya belum mencoba sendiri, saya tidak akan puas dan itung-itung juga menambah pengalaman.

Keesokan harinya, saya datang ke kantor tersebut. Saya bertanya di salah satu satpam yang ada di depan gedung arah perusahaan Agrogroub dimana. Tapi anehnya satpam tersebut tidak tau tentang perusahaan ini dan yang ia tau adanya Agrodana. Lalu saya di beri tau kalau Agrodana ada di lantai 2. Setelah saya di lantai 2, saya bertanya dengan receptionist kalau saya melamar pekerjaan dan menemui salah satu karyawan tersebut. Setelah menunggu beberapa lama, saya di arahkan masuk ke dalam ruangan kaca dan di sana saya disediakan soal psikotest. saya waktu itu heran karena saya tidak di beri tau kalau ada psikotest. Oke saya kerjakan testnya dan kebetulan testnya materinya sama dengan salah satu materi kedinasan yang pernah saya coba dan testnya sangat mudah. Lalu setelah test, saya menyerahkan lamaran dan disuruh untuk datang lagi ke kantor besok untuk interview.

Keesokan harinya saya datang ke kantor dan waktu itu saya di perlihatkan hasil test saya. Saya begitu kaget dengan hasil test saya 72 dan untuk jadi staff admin minimal harus 85. Di pikiran saya, saya mengerjakan soal tersebut dengan baik dan soalnya lebih mudah dari soal kedinasan yang pernah saya kerjakan dulu. Okelah tidak apa-apa mungkin karena saya sudah tidak pernah belajar jadi maklum lah. Karena saya di sini tidak memenuhi persyaratan, saya di beri tawaran oleh manager yang mewawancariai saya. ia berkata, "Maaf nilai kamu kurang untuk jadi Staff Admin di sini. Tapi saya tidak melihat kamu dari hasil test ini saja tapi yang saya butuhkan adalah praktek kinerja kamu. Karena kamu jauh-jauh ke sini cari kerja masa kamu nggak dapet kerja maka saya tawari kamu untuk training dulu selama 2 bulan dan tugasnya hanya telfon tok nanti gaji kamu per bulan 1,2 juta. kira-kira kamu mau nggak?", tanya manager tersebut. Lalu karena saya belum punya pengalaman saya menerima tawaran tersebut. Lalu saya bertanya lagi "Apakah setelah saya lepas training, saya bisa di Staff Admin?", "YA BISA", jawab manager tersebut. Saya pikir-pikir lumayan buat menambah pengalaman. Setelah itu saya di suruh untuk mulai training minggu depan hari senin dan masuk jam 09.00 WIB.


2. Masa Training Bulan Pertama

Setelah saya menunggu beberapa hari untuk training, akhirnya datang juga hari itu dimana saya berangkat ke kantor jam 08.30 dan suasana di kantor saat itu masih sangat sepi. lalu aku dipersilahkan duduk di salah satu meja dan lebih tepatnya di team orang yang merekrut ku. Awalnya saya sangat tegang karena di pikiran saya saat itu adalah training jadi setiap mengambil keputusan harus berhati-hati. Setelah itu saya di ajari untuk telfon. Saya kira saya di suruh telfon untuk membuat jadwal perjanjian dengan atasan kami tapi ternyata beda. Saya disuruhh telfon untuk mengajak ketemu dengan atasan saya untuk menawarkan investasi. Dan sebelum itu saya di ajari untuk mencari database yaitu data nomor telfon suatu usaha atau perusahaan, alamat, dan lain sebagainya untuk di telfon. Selanjutnya saya disini mencari database lewat internet. Lalu setelah istirahat, saya di beri surat perjanjian seperti kontrak tapi bukan kontrak yang berisi tentang aturan-aturan sebagai Account Executive (AE) atau bisa di sebut marketing dan pemegang akun. Saya begitu kecewa dengan perjanjian tersebut karena saya malah di pindahkan di marketing. Tapi saya tetap optimis setelah masa training pasti saya di tempatkan di Staff Administrasi.

Bepberapa hari kemudian, Saya di beri aturan supaya gaji keluar 1,2 juta maka saya harus cari APP sebanyak 10 per bulan. APP adalah ketika kita menelfon atasan dari perusahaan untuk mau atur janji ketemu atasan saya maka saya mendapat 1 poin APP dan APP tidak boleh sembarangan, targetnya adalah orang yang mampu dan dana ready sekitar 50jt. Nah saat itu saya sangat kesusahan mencari APP karena tidak semua orang mau untuk ditemui, apa lagi ini adalah musim pandemi yang membatasi seseorang unruk berinteraksi. Setiap pagi saya disuruh untuk membuat berita fundamental dan tekhnikal serta menghitung pivot point, resistant, dan support untuk dibagikan ke nasabah atau calon nasabah dan grub. Batasnya adalah mulai jam 05.00-06.00. Bagi yang terlambat akan dikenai hukuman dan dimarahi.

Lalu setelah beberapa hari sampai akhir bulan sekitar tanggal 20an, APP saya hanya ketemu 5 point saja dan yang 5 point lagi saya mencari APP di kampung halaman saya sendiri dan bersosialisasi sendiri dengan orang yang mungkin saya kenal dan dana ready sekitar 50 juta dan kebetulan hari itu adalah ketika libur tahun baru jadi saya ada waktu lebih untuk mencari APP di kampung halaman saya. Setelah saya cari APP genap 10, saya menunggu hari gajian yaitu tanggal 6 awal bulan Januari. Ketika itu saya kecewa yang kedua kalinya karena gaji saya hanya keluar 950. Lalu saya bertanya kepada atasan saya mengenai gaji saya dan saya dijelaskan tentang sistem gaji di situ. saya di jelaskan kalau gaji absen saya 700rb + per APP 25rb dan minimal 10 APP supaya gaji bisa keluar. karena saya mendapat 10 APP, maka saya mendapat total gaji 950rb. Saya sangat kecewa karena saya tidak diberi tahu secara transparan tentang sistem yang ada di sini. Lalu karena saya ingin mencari pengalaman disini, maka saya menunggu untuk bulan yang kedua.


3. Bulan Kedua

Di bulan kedua ini adalah bulan yang banyak sekali terdapat keganjalan dan cara kerja para atasan saya. Pada bulan awal saya tanya ke rekan kerja saya apakah di sini benar ada lowongan staff admin apa tidak karena saya amati admin di kantor saya cuma satu dan mereka yang baru melamar kerja, hasil test nya di salahkan dan sengaja di nilai 72 sama assisten manager saya. Lalu kata teman saya lowongan staff admin itu tidak ada dan itu cuma buat pemanis saja supaya saya mau bekerja di sini. ini Adalah kekecewaan saya yang ketiga kalinya. tapi karena saya kerja di bulan kedua ini sudah hampir pertengahan bulan, maka saya ingin melanjutkan bekerja di sini dan karena waktu itu APP saya hanya dapat 3 dan saya dipaksa untuk closing R1 saya (orang yang saya kenal dekat seperti kerabat), maka saya berkeinginan untuk closing 10Jt dari kakak sepupu saya. Karena kata asmen saya, apabila saya closing mini meskipun 10jt, maka saya akan langsung keluar gaji saya 1,2jt + total komisi per transaksi. Closing di sini adalah ketika saya telah mendapatkan nasabah untuk menginvestasikan dananya di perusahaan kami.

Ini adalah bagian penting dari topik ini. Disini saya dipaksa untuk closing kerabat saya lalu saya tawarkan melalui Whatsapp dan langsung saya sambungkan ke manager saya. Kakak sepupu saya hanya di tawarkan keuntungan per bulan 10% dan bisa lebih dari 10% karena ini adalah bulan januari ketika emas lagi momentnya naik (kata manager saya). Lalu kakak sepupu saya percaya begitu saja karena ada saya yang bekerja di sini dan ia juga percaya dengan keluarganya sendiri. Pada saat sudah di transfer dan kami mengurus registrasi online, kami mengisi data-data diri kakak sepupu saya dan harusnya itu di isi oleh kakak sepupu saya sendiri tapi sama managerku suruh isi kami saja. Lalu saat mengisi nomor telfon, manager saya dan assistennya bingung mengisi pakai nomornya siapa karena katanya nanti akan di tanyai langsung oleh Agrodana pusat apakah di janjikan keuntungan, apakah sudah tau resiko bertransaksi di sini kalau resiko bertransaksi di sini sangat besar dan lain sebagainya. Lalu saya bilang apakah sebaiknya kakak saya diberi tahu kalau setiap transaksi di sini ada kerugiannya. Tapi saya tidak boleh memberi tahu kakak sepupu saya dan saya disuruh tutup mulut tentang adanya resiko kerugian, karena mereka takut kalau tidak jadi closing di sini. Saya waktu itu juga agak curiga dengan cara kerja manager saya melanggar aturan dari perusahaan.

Dan parahnya lagi mereka menyuruh saya mengisikan nomor HP saya di akun kakak saya, dan rencananya nanti kalau ada telfon dari pusat biar asmen saya yang mengangkatnya. saya di situ juga agak muak daengan cara kerja mereka, tapi karena saya di tuntut untuk menuruti apa kata atasan saya, maka saya percaya saja dengan mereka dan mereka tidak pernah menyebutkan adanya kerugian saat transaksi jadi saya pikir hal itu akan baik-baik saja. Keesokan harinya, saya melanjutkan registrasi online. Setelahnya, ditampilkan halaman yang menyuruh saya untuk membaca aturan-aturan yang ada di agrodana saat saya meregistrasikan akun kakak sepupu saya. Waktu itu saya hanya membaca sekilas karena saya disuruh untuk cepat mencentang persetujuan dari aturan-aturan tersebut. Registrasi selesai, saya diberi tahu untuk bertransaksi hari senin tanggal 18 Januari 2021.

Hari senin telah tiba. Pada saat itu sebelum market buka, saya membuat berita seperti biasa. Setelah membuat berita, saya menunggu pukul 06.00 untuk mulai bertransaksi. Dan waktu itu saya di suruh untuk bertransaksi sendiri dan di arahkan untuk ambil opsi "buy". sebenarnya saya ragu memilih opsi tersebut karena pivot pointnya lebih kecil dari harga open dan seharusnya memilih opsi "sell". Tapi karena manager saya menyuruh saya pilih opsi buy, saya percaya dengan pilihan manager saya yang mengikuti moment "buy" saat bulan januari sampai imlek dan karena mereka lebih berpengalaman. setelah jam 06.00, saya open posisi BUY dan saya terus memantau. tapi prediksi manager saya salah, harga saat itu turun drastis sampai Stop Loss saya kena dan saya rugi 440rb saat itu juga. lalu akun saya saya serahkan ke manager saya karena waktu itu saya tidak bisa memantau dan saya siap-siap untuk berangkat kerja. Lalu manager saya membuka Sell Limit di harga agak tinggi dan waktu itu saya tidak tau apa itu sell limit. Lalu saat saya ada tugas ke luar, saya lihat harga market. Dan benar saja, harga emas naik drastis dan ekuitasku berkurang 2jt lalu manager saya langsung CutLoss. Jadinya waktu itu saya rugi 2jt lebih dalam satu hari. Saya begitu lemas saat itu dan tidak lagi bersemangat dalam bekerja.

Pada sore harinya, saya dipanggil oleh atasan manager saya untuk membicarakan teman saya yang resign masal dari sini dan membicarakan tentang kerugianku tadi pagi. Pada saat membicarakan teman saya yang resign, saya waktu itu juga kaget kok mereka resign secara tiba-tiba. Pada waktu itu atasan manager saya mengolok-olok dan memaki-maki teman saya yang resign dan saya di kasih tau kalau saya jangan seperti mereka yang resign. Lalu setelah itu, saya di beri semangat untuk jangan panik ketika akun saya rugi. Katanya, manager saya akan memperbaiki akun saya supaya saldonya kembali ke awal dan saya di suruh menunggu 4 hari biar saldonya kembali. Tapi lagi-lagi saya juga yang di suruh transaksi dan waktu itu saya hanya di arahkan. meski untung sedikit demi sedikit tapi dengan saya sendiri yang bertransaksi, itu juga mengandung resiko yang besar karena saya belum berpengalaman di bidang trading. Jadi saya harus belajar saat itu juga. Pada saat saya bertransaksi sesuai keinginan saya sendiri, saya malah di marahi sama asmen saya katanya kalau market malam itu bahaya tapi menurut saya, malam itu harga anjlok dan ini kesempatan saya untuk ambil harga BUY. Tapi karena saya kena marah asmen saya, saya menutup harga dan untung hanya 120rb. Apabila saya waktu itu belum tutup harga, mungkin saya bisa untung lebih banyak dari itu.

Setelah kami kumpulkan profit-profitnya, sekarang saldo saya hampir kembali seperti awal, yaitu 9,2jt. Lalu pada saat di kantor, saya di beri lembaran kertas yang isinya tentang aturan-aturan penting yang harus di patuhi oleh pemegang akun sekaligus maarketingnya (Account Executive). Di situ terdapat beberapa pelanggaran yang telah di lakukan oleh manager saya dan saya pun juga ikut melanggar aturan. Tapi pada saat itu saya tidak mengetahui apa-apa. point pentingnya dilanggar yaitu:

  1. Apaila belum mempunyai sertifikat resmi dari BAPPEBTI, maka orang tersebut tidak boleh bertransaksi.
  2. Tidak boleh menyebarkan informasi rahasia dari akun nasabah seperti email dan password. (Dan waktu itu manager dan asmen saya juga mempunyai akun kakak sepupu saya).
  3. Sebelum bertransaksi, Perusahaan harus memberitahukan resikonya dan harus ada dokumennya tentang resiko saat bertransaksi di pasar bursa serta perusahaan harus membuat perjanjian dengan nasabah akan adanya resiko.
Lalu saya disuruh untuk menandatangani aturan tersebut. Pada saat itu saya sangat menyesal dan saya mulai saat itu tidak mau menjalankan transaksi dengan tangan saya sendiri dan saya selalu meminta agar transaksi dilakukan oleh atasan saya dan memberitahukan kepada saya kalau akan bertransaksi karena sesuai aturan, saya belum boleh untuk bertransaksi.

Pada saat saya kembali dari tugas di luar, atasan dari team lain sakit dan kantor menyuruh mulai besok semua harus rappid dan apabila belum rappid test maka tidak boleh masuk ke kantor. Lalu besoknya saya test rappid di puskesmas Taman, tapi mereka tidak bisa rappid pribadi. Lalu saya pulang ke kost dan sambil mencari rumah sakit atau klinik untuk rappid. Besoknya pada hari jum'at saya rappid di klinik dan pada saat itu saya tidak memantau sama sekali marketnya. Setelah saya rappid, saya kaget kalau akun kakak saya di transaksikan dengan membuka harga BUY di harga 1868. Pada saat itu masih rugi dan pada sore harinya saya pulang kampung dan akun saya sudah ambil posisi kuncian. Tapi hari demi hari dan kuncian di buka dan di tutup tapi hasilnya malah ekuitas saya makin menurun menjadi 5jt. lalu saya tunggu berhari-hari sampai akun saya diperbaiki oleh manager saya tapi kenyataannya akun saya di biarkan begitu saja. Saya sangat depresi waktu itu dan saya tidak berniat sama sekali untuk bekerja sampai-sampai atasan saya bilang kalau akhir-akhir ini kinerja saya semakin menurun. Bagaimana tidak? akun saya yang rugi gara-gara mereka bukannya di perbaiki dulu tapi malah saya di suruh untuk closing lagi supaya margin mereka naik dan mereka bisa naik jabatan. Mana bisa gitu? ngurus satu akun saya saja nggak pecus apa lagi ngurus akun orang lain lagi. Malah hancur 2 akun bisa-bisa. Dan lagi saya mendapat cerita dari teman saya kalau akun teman saya ludes 200jt. Kejadiannya hampir sama seperti saya. Lalu teman-teman saya juga di ceritakan kalau akun teman saya rugi 200jt lalu teman-teman saya juga berhati-hati dalam bekerja disana supaya tidak terjebak.

Setelah itu saya tidak lagi bersemangat bekerja. Lalu pada saat gajian, saya hanya di transfer 720rb. Lalu saya protes ke manager saya, tapi manager saya sedang sakit katanya dan tidak bisa masuk kerja pada saat itu. Mau tidak mau saya protes ke asmen saya dan pada waktu itu saya berdebat dengan asmen saya dan kedua senior team lain. Saya protes menagih janji kalau saya closing, gaji saya keluar 1,2jt + transaksi, meskipun APP saya tidak 10. Pada saat itu asmen saya tidak bisa berkata apa-apa dan dia tidak mau bertanggungjawab dengan omongannya. lalu saya di jelaskan oleh senior dari team lain kalau sebenarnya sistem gajinya sama yaitu 700rb untuk absen, 25rb untuk APP nya dan 380rb untuk komisi closing tapi dengan syarat harus lepas training dan saya kena potongan tidak masuk karena rappid, jadi totalnya 720rb. Waktu itu saya sangat marah karena saya telah di tipu oleh asmen saya sendiri dan saya menuntut untuk mempertanggungjawabkan omongannya lalu saya juga membahas tentang akun kakak sepupu saya kalau jangan di biarkan begitu saja, mereka harus bertanggungjawab. Tapi mereka tidak mau bertanggungjawab dan mereka sama sekali tidak bisa menjawab apapun yang saya katakan. Lalu, keesokan harinya, manager saya masuk kerja dan saya saat itu saya di panggil ke ruangan manager saya. Saya berdebat panjang dengan manager saya karena saya ingin mereka bertanggungjawab atas kerugian akun kakak saya. Tapi mereka tidak mau dan mereka bodo amat dengan akun saya. Dan saya terus menuntut mereka untuk bertanggungjawab, tapi karena saya di suruh keluar oleh atasan manager saya, maka saya mengundurkan diri dari perusahaan itu. Saat itu teman-teman saya baik yang sudah resign maupun belum, kabar baiknya mereka akan selalu mendukung kami sampai masalah ini tuntas, karena diam-diam, teman saya juga tidak suka dengan sistem kerja atasan saya dan mereka sudah membencinya dari dulu.
